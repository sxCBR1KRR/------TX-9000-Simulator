# Упатство за стартување на веб-сајтот

Бидејќи оваа верзија на проектот користи модерен развој со **Vite** и **JavaScript модули**, не можете едноставно да го отворите `index.html` фајлот во прелистувачот. За сајтот да работи правилно (да се вчита 3D моделот и скриптите), потребен е локален сервер.

Еве ги чекорите како да го стартувате проектот на вашиот компјутер:

### 1. Инсталација на Node.js
Прво, морате да имате инсталирано **Node.js** на вашиот компјутер. Ако ја немате, преземете ја од официјалниот сајт: [nodejs.org](https://nodejs.org/). (Препоникајте ја LTS верзијата).

### 2. Отворање на проектот
Отворете го вашиот омилен едитор за код (на пример: **WebStorm**, **VS Code** или **Sublime Text**) и отворете го фолдерот:
`threejs-glb-app`

### 3. Инсталирање на зависностите (Dependencies)
Отворете го терминалот (Terminal) во вашиот едитор и напишете ја следната команда за да се инсталираат сите потребни библиотеки (Three.js, GSAP итн.):
```bash
npm install
```

### 4. Стартување на локалниот сервер
Откако ќе заврши инсталацијата, стартувајте го проектот со следната команда:
```bash
npm run dev
```

### 5. Преглед во прелистувачот
По извршувањето на командата, во терминалот ќе се појави линк (најчесто `http://localhost:5173/`). Кликнете на тој линк или копирајте го во вашиот прелистувач и ќе можете да го видите симулаторот во работа!
